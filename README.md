# Proyecto Final de Programación IV: Portafolio personal profesional

## Descripción
Este es mi portafolio personal, un proyecto final de la cursada de programación IV hecho para mostrar mis trabajos destacados.

## Tecnologías utilizadas
* Astro: La base de todo.
* Tailwind CSS: Para el diseño.
* GSAP (GreenSock): Para las animaciones.
* Formspree: Usado para la funcionalidad de formularios de contacto.
